# Bankroll Lab Design System

## Visual truth

The selected reference is `design/reference-panel-control.png`.

## Tokens

- Canvas: deep graphite navy, nearly black.
- Surface: a slightly lighter blue-black.
- Border: thin slate-blue hairlines.
- Primary text: crisp off-white.
- Secondary text: desaturated blue-gray.
- Accent: cool cyan-blue for selection and charts.
- Positive: green, used only for gains.
- Negative: red, used only for losses and risk.
- Warning: amber, used only for alerts.
- Radius: restrained 12px surfaces, 10px controls.
- Typography: humanist sans for UI, tabular numerals for money and metrics.

## Component families

- Metric panels use fine borders, flat elevation and clear numeric hierarchy.
- Lists are grouped surfaces with row dividers, not collections of floating cards.
- Navigation is fixed at the bottom with five equal targets and a cyan selection rail.
- Primary forms open as mobile drawers with large inputs and explicit confirmation.
- Charts use blue strokes/fills, minimal axes and short labels.

## Density

The primary mobile viewport is 390 x 844. The interface is information-rich but never cramped: 16px page gutters, 12px section gaps, 44px minimum interactive targets and safe-area padding.
